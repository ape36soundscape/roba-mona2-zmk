roBa × moNa2
Compatible Firmware

ふたつの作品をつなぐ、やさしい ZMK。

はじめる

コンセプト

kumakey さんの「roBa」ハードウェアと、pooh_polo さんの「moNa2」ファームウェア。 どちらも大切な作品です。このプロジェクトは、その魅力をそっとつなぐカスタム ZMK。 見た目（名前）は moNa2、中身（配線）は roBa。 ちょっと実験的ですが、楽しく、やさしく、遊べることを目指しています。

特徴

ハード

roBa（kumakey）

MCU

Seeed XIAO nRF52840 ×2（左右）

通信

BLE スプリット（左＝Central / 右＝Peripheral）

センサー

PMW3610（右トラックボール）

スイッチ

Choc V2（ホットスワップ）

LED

なし（ミニマル）

FW

ZMK + PMW3610 ドライバ

表示名

moNa2 / moNa2-P（BLE/USB 名のみ）

はじめる（GitHub Actions で UF2 を自動生成）

  1. このページのソース（ZIP）を GitHub にそのままアップロード（リポジトリを作成）
  2. Actions が自動で走り、left.uf2 / right.uf2 が生成されます
  3. Artifacts からダウンロードして、XIAO nRF52840（左右）にコピー
もう少し詳しく 
# 例: ZMK を取得し、zmk-config を配置（ローカルビルドの場合）
west init -m https://github.com/zmkfirmware/zmk zmk
cd zmk && west update
cd .. && mv zmk-config ./zmk-config

# 右（Peripheral）
west build -p -b seeeduino_xiao_nrf52840 build/right --   -DSHIELD=roba_compat_mona_right -DOVERLAY_CONFIG=config/split_right.conf

# 左（Central）
west build -p -b seeeduino_xiao_nrf52840 build/left --   -DSHIELD=roba_compat_mona_left -DOVERLAY_CONFIG=config/split_left.conf

※ VID/PID は変更しません。デバイス名のみ「moNa2 / moNa2-P」にしています。
